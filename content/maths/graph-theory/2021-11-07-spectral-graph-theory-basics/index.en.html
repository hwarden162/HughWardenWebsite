This is a stub for an article currently in the process of being written. For the interested reader, this is a brilliant mini course on the subject backed up by the spectral graph theory wikipedia page.

The basics of the subject is that if you have a graph you can represent it using many different matrices. You can find the eigenvalues and eigenvectors of these matrices and depending on which matrix you use these can have special properties. In the end what this means is that you can represent a graph just as a set of numbers that will contain information about the graph.
